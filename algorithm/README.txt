ジャンケン

変数 $player と $rival には、
ジャンケンの履歴が文字列で入っています
プレイヤー側が何勝何敗したかを求めて、
勝利数：$win
敗北数：$lose
それぞれの変数に結果を格納してください 

<数字の意味>
0:グー
2:チョキ
5:パー

（例）
$player = "050020";
$rival = "500025";

<結果>
$win => 1
$lose => 2
